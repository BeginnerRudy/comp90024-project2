# Backend

## How to run
### Ubuntu
1. ```cd backend```
2. ```python3 -m venv venv```
3. ```. venv/bin/activate```
4. ```pip install -r requirements.txt```
5. ```pip install flask```

## generate requirements.txt
1. ```python3 -m venv venv```
2. ``` pip freeze > requirements.txt```
`
